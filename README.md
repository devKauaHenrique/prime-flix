# 🎬 Prime Flix

Uma aplicação web inspirada em plataformas de streaming, desenvolvida para praticar consumo de APIs, manipulação de dados e construção de interfaces modernas utilizando JavaScript.

O projeto permite explorar informações sobre filmes, visualizar detalhes e navegar por um catálogo dinâmico de conteúdos.

## 🚀 Demonstração

🔗 Acesse o projeto online:

https://prime-flix-iota.vercel.app

## ✨ Funcionalidades

* Listagem dinâmica de filmes
* Exibição de detalhes dos filmes
* Busca de conteúdos
* Navegação intuitiva
* Layout responsivo
* Integração com API externa

## 💻 Como Executar Localmente

1. Clone o repositório

```bash
git clone https://github.com/devKauaHenrique/prime-flix.git
```

2. Entre na pasta

```bash
cd prime-flix
```

3. Instale as dependências

```bash
npm install
```

4. Execute o projeto

```bash
npm run dev
```

5. Abra no navegador

```bash
http://localhost:5173
```
